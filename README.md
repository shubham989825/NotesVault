# React + Vite

## 🚀 NotesVault

NotesVault is a full-stack note-taking web application that allows users to securely create, manage, and organize personal notes online.

It is built using the MERN stack and deployed with modern cloud platforms.

---

## 🌐 Live Demo

🔗 Frontend (Vercel): https://notes-vault-mu.vercel.app 
🔗 Backend (Render): https://notesvault-ji1u.onrender.com

---

## ✨ Features

- 🔐 Secure User Authentication (Register / Login)
- 📝 Create, Update, Delete Notes
- 🔒 JWT-based Authorization
- 🔑 Password Hashing with bcrypt
- 📱 Fully Responsive Design
- ⚡ Fast and optimized with Vite
- ☁️ Cloud deployment (Vercel + Render)

---

## 🛠 Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- React Router DOM
- Axios

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication
- bcrypt
